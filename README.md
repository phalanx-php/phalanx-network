# Phalanx Network

Network discovery, probing, and device-management tasks for Phalanx. Network packages subnet scans, port checks, wake-on-LAN, and host/service inspection into composable runtime tasks.

Part of the [Phalanx framework](https://github.com/phalanx-php/phalanx).
